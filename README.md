# Smart Video Downloader - Frontend

This is the frontend for the Smart Video Downloader application.

## Features
- Modern UI with Dark/Light theme
- Auto video size detection
- Multiple platform support (YouTube, Instagram, TikTok, etc.)
- Responsive design

## Setup
1. Clone this repository
2. Open `index.html` in browser
3. Update `API_BASE_URL` in script to point to your backend
